 The AI examination system is developed with Vue 3 for the front end and Spring Boot 3 for the back end. 
 It utilizes MySQL, Redis, Kimi AI, and MinIO. The system includes both a student platform and a management platform. 
 The student platform mainly features three functions: examinations, learning, and score inquiry. 
 The core examination function supports AI intelligent approval, real-time timing, and automatic answer saving.
 The management platform mainly includes functions such as question management, batch operation of questions, test paper management, examination records, and system management. 
 Redis cache optimization, batch operation, and N+1 problem optimization are applied in question management.
